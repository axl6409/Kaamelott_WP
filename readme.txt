== Description ==

This plugin display quotes of the famous french TV Show __*Kaamelott*__. 

The quotes are display on the top of the administration panel. 
It was a project for my training in PHP, when i learn 'array' in PHP.

This plugin is based on the 'Hello Dolly' plugin.


== Installation ==

For the installation :

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
